# Symptom-to-Diagnosis-Probabilistic-Engine
This project implements a Bayesian probabilistic engine in R that estimates the likelihood of different diseases given a set of reported symptoms. Rather than producing a single diagnosis, the model outputs a ranked probability distribution, reflecting uncertainty in real-world clinical reasoning.
